## 质量检查清单

写作时每章必须检查：

- [ ] 故事有真实场景感，不是干巴巴的案例
- [ ] 主角的行为和身份一致
- [ ] 理论知识有框架/模型/方法论
- [ ] 实践部分有可执行的操作步骤
- [ ] 包含具体的Prompt示例
- [ ] 每章字数在8000-12000字之间
- [ ] 语言风格口语化，像跟读者聊天
- [ ] 有至少2个彩蛋
- [ ] 结尾有下一章预告
