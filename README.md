## 📖 关于本书

这是一本写给程序员的AI实战指南。

不是教你"学AI"，而是教你**用AI偷懒**——让AI写代码、做测试、写文档，你只管喝咖啡、想架构、早点下班。

从第一次用AI被惊艳到，到搭建团队级AI工程平台，这本书会陪你走完这段**从打工人到指挥官**的进化之路。

---

## 🎯 适合谁读

- **前端/后端/全栈开发**：想让AI帮你写CRUD、补全代码、改bug
- **测试工程师**：想让AI生成测试用例、写自动化脚本
- **运维工程师**：想让AI写Shell、配Docker、搭CI/CD
- **产品经理**：想让AI写PRD、画流程图、分析竞品
- **技术负责人**：想给团队引入AI工程化，提升整体效率

**不适合**：想深入了解AI原理的算法工程师（本书只讲怎么用，不讲怎么造）

## 📝 写作风格

每章统一采用以下结构：

1. **故事开篇**（1-2页）：具体人物的真实场景，有冲突、有转变
2. **理论知识**（3-5页）：方法论和原理，不枯燥
3. **实践案例**（3-5页）：具体操作步骤，可执行
4. **本章交付物**（1页）：完成后可验收的成果
5. **行动清单**（半页）：课后作业
6. **本章彩蛋**（半页）：有趣的小知识
